# The Thing Constant Is Change: Three Tech Trends That Will Shape Our Future

Presented at 3|SHARE's EVOLVE'15 - The Adobe Experience Manager Community Summit, this keynote explores three transformative technology trends that are reshaping the digital landscape: mobile dominance, wearables, and the Internet of Things.

## Key Trends

### Mobile Dominance

- **Prediction from 2008**: "Mobile to overtake fixed Internet access by 2014" - Mary Meeker
- **Reality**: Mobile has become the dominant platform for internet access
- **Impact**: Mobile design needs to be more than just responsive design

### Wearables Explosion

- **Prediction**: "Wearables set for explosive growth" - CNET
- **Platforms**: Android for Wearables, Apple Watch OS, Tizen
- **Integration**: Connecting with mobile devices and cloud services

### Internet of Things (IoT)

- **Scale**: "The Internet of Things Is Far Bigger Than Anyone Realize" - Wired Magazine
- **Platforms**: Brillo (Google), Homekit (Apple), SmartThings
- **Impact**: Everything connected and integrated, creating BIGGER DATA

## A Toolbox for Success

### Mobile First Approach

- Progressive Enhancement builds a great mobile experience then builds on that for desktop
- Graceful Degradation builds a great desktop experience then waters it down for mobile
- Mobile has a greater international reach

### Hybrid First Development

- Utilize a common toolset - HTML, CSS, and JavaScript
- PhoneGap supports all major mobile platforms and some wearable platforms
- Large library of plugins
- Mobile UI Frameworks - Ionic Framework, React Native, Kendo UI, Famo.us
- Create custom plugins for native functionality
- Go native as a last resort

### JavaScript First Strategy

- Ubiquitous language, found everywhere
- Simplifies team skill set, one language, whole stack
- Isomorphic coding
- Example Stack: Sightly, Angular, Ionic

### AEM and PhoneGap Enterprise

- **AEM Benefits**:

  - Components
  - Experiences
  - Content Management
  - Asset Optimization - Images, JavaScript and CSS
  - Analytics and other integrations

- **PhoneGap Enterprise Benefits**:
  - Leverage AEM's Content Authoring
  - Leverage AEM's Integrations
  - Push Messaging - Adobe Campaign
  - Testing Tools and Workflows
  - Build And Publish Workflow Automation
  - Over the Air Updates

## WWCW: What Would Chuck Want?

Design principles for building better mobile experiences:

1. **Simplicity**: Achieved when there is nothing left to take away. No second clicks when one will do.

2. **Goal Oriented**: He has a job to do. Don't get in his way. The goal should be clear.

3. **Content In Context**: When he wants it. Where he wants it. Because he wants it.

4. **Actionable Data**: Actionable insights. Not what the data is but what it means. What can he do about it.

5. **No Excuses**: Chuck always gets what he wants.

## Conclusion

The digital landscape is constantly evolving with new platforms, devices, and user expectations. By embracing a mobile-first, hybrid-first, and JavaScript-first approach, and leveraging tools like AEM and PhoneGap Enterprise, organizations can create experiences that meet users' needs across all platforms and devices.
